# 🎉 8살 딸을 위한 생일 앱! 🎉

8살 딸의 특별한 날을 위해 특별히 디자인된 재미있고 인터랙티브한 생일 축하 앱입니다!

## 기능

### 🎈 인터랙티브 게임
- **풍선 터뜨리기 게임**: 알록달록한 풍선을 클릭해서 터뜨리고 폭발하는 모습을 보세요!
- **촛불 끄기 게임**: 8개의 생일 촛불을 모두 클릭해서 불어 끄세요

### 🎊 특별 효과
- 깜짝 선물 버튼을 클릭하면 색종이 애니메이션
- 부드러운 애니메이션으로 떠다니는 풍선들
- 화면 곳곳에 나타나는 랜덤 반짝임
- 통통 튀는 생일 제목

### 💝 생일 축하 메시지
- 격려의 메시지가 담긴 아름다운 축하 카드들
- 아이들이 좋아하는 알록달록하고 친근한 디자인
- 태블릿과 휴대폰에서도 잘 작동하는 반응형 레이아웃

## 사용 방법

1. 웹 브라우저에서 `index.html`을 열어주세요
2. "깜짝 선물을 위해 클릭해!" 버튼을 클릭하면 색종이와 생일 메시지가 나타납니다
3. 풍선을 클릭해서 풍선 터뜨리기 게임을 즐기세요
4. 각 촛불을 클릭해서 생일 촛불을 불어 끄세요
5. 알록달록한 애니메이션과 생일 축하 메시지를 즐기세요!

## 기술적 세부사항

- 순수 HTML, CSS, JavaScript (외부 의존성 없음)
- 모바일 친화적인 반응형 디자인
- 아이들이 좋아하는 재미있고 친근한 한국어 폰트
- 부드러운 애니메이션과 전환 효과
- 점수 추적 기능이 있는 인터랙티브 게임

## 맞춤 설정

다음과 같이 앱을 쉽게 맞춤 설정할 수 있습니다:
- `styles.css`에서 색상 변경
- HTML에서 더 많은 생일 축하 메시지 추가
- `script.js`에서 게임 수정 또는 새 게임 추가
- 아이의 이름을 추가해서 더 개인적인 경험 만들기

특별한 8번째 생일을 더욱 기억에 남게 만들어주는 완벽한 앱입니다! 🎂✨
